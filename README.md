# Muzika 🎵

A lightweight, modern cross-platform mobile application built with Flutter that allows users to seamlessly search for YouTube videos, extract/download their audio files in `.m4a` format, and play them offline anytime, anywhere.

---

## 🚀 Features

* **YouTube Search:** Look up any video directly within the app using the YouTube search functionality.
* **Audio Extraction:** Fetch and download high-quality `.m4a` audio files instantly.
* **Offline Media Player:** Built-in local audio player to listen to your downloaded tracks without an internet connection.
* **Background Playback:** Keep the music running even when the app is minimized or the screen is locked.
* **Clean UI/UX:** A minimalist, fast, and responsive interface designed using Flutter.
